Thư mục entity chứa các class đại diện cho bảng trong database, được ánh xạ thông qua JPA/Hibernate với các annotation như @Entity, @Table, @Id, @Column. Entity giúp ánh xạ giữa object trong Java và table trong database (ORM). Entity chỉ nên dùng để biểu diễn dữ liệu lưu trữ trong DB, không dùng trực tiếp để trả về client tránh lộ thông tin nhạy cảm hoặc gây lỗi khi serialize JSON.
