Thư mục dto (Data Transfer Object) dùng để trao đổi dữ liệu giữa các tầng và với client chia thành request (dữ liệu gửi lên server) và response (dữ liệu trả về client). DTO giúp kiểm soát chính xác những field nào được nhận và trả về, tránh lộ dữ liệu nhạy cảm (ví dụ password), giảm kích thước response và tránh lỗi vòng lặp JSON do quan hệ entity.
