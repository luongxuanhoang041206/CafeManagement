Thư mục controller chứa các class đóng vai trò là điểm vào (entry point) của hệ thống, tiếp nhận HTTP request từ client (Frontend, Postman, Mobile, v.v.) thông qua các annotation như @RestController, @RequestMapping, @GetMapping, @PostMapping... 
Controller có nhiệm vụ nhận dữ liệu đầu vào, validate, sau đó gọi xuống tầng service để xử lý nghiệp vụ và trả về response cho client.
